# Cadence

Cadence is a static focus rhythm planner for developers, builders, learners, and operators.

## What it does

- Turns your day context into a realistic focus schedule
- Estimates deep-work capacity, switching drag, and recovery debt
- Suggests block lengths, breaks, and a close-out ritual
- Syncs state into the URL for easy sharing

## Deploy URL

- https://cadence.entazis.dev

## Why this app

AI tools are making people faster and more fragmented at the same time. Cadence is a small counterweight: a daily rhythm planner for sustainable output instead of productivity theater.
